<h1> Instructions: </h1>
<ul>
	<li> Configure your Xbee via Xctu </li>
	<li> Start create your firmware from emptyTemplate</li>
</ul>

<h2>How:</h2>
<p>
-Open Xctung connect your XBEE and click on Load Profile.
-Click edit profile
-Disconnect the XBEE
-Connect XBee to your Arduino
-Duplicate emptyTemplate and rename it with the name you want
-Use ApioCodify to write the sketch for example see the example folders.
</p>
